# 泊松回归 (Poisson Regression)

## 核心定义
**泊松回归** 是用于建模**计数数据 (Count Data)** 的标准模型。
- **响应变量 $Y$**：非负整数 ($0, 1, 2, \dots$)。
- **例子**：每小时路过的车辆数、一篇文章中的错别字数、一年内的台风次数。

## 数学模型
### 1. 链接函数：Log
我们假设 $Y$ 服从泊松分布，其均值为 $\lambda$。我们使用 **对数链接函数 (Log Link)**：
$$\log(\lambda(X)) = \beta_0 + \beta_1 X_1 + \dots + \beta_p X_p$$

或者反过来写（预测形式）：
$$\lambda(X) = e^{\beta_0 + \beta_1 X_1 + \dots + \beta_p X_p}$$
* **为什么用 Log？** 因为计数 $\lambda$ 必须大于 0。$e^{\dots}$ 保证了无论 $\beta X$ 是多少，预测结果永远是正数。

### 2. 核心假设：均值等于方差
泊松分布有一个极强的假设：
$$E(Y) = Var(Y) = \lambda$$
这意味着：**事件发生得越频繁（均值越大），其波动（方差）也应该同比例变大。**

## 为什么不用线性回归？
如果你用线性回归 ($Y = \beta X$) 预测计数：
1.  **负数风险**：可能预测出“明年台风次数为 -2.5 次”，这不合逻辑。
2.  **方差恒定**：线性回归假设方差不变，但这不符合计数数据的特性（高均值通常伴随高方差）。

## 关联笔记
- 如果数据的方差远远大于均值（违反了核心假设），请使用 [[负二项回归]]。
